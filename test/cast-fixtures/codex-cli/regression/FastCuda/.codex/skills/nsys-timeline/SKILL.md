# Nsight Systems Timeline

Use this skill for stream overlap, launch overhead, and end-to-end timeline
analysis with Nsight Systems.

## Workflow

1. Run `scripts/perf/profile-nsys.ps1`.
2. Save timeline artifacts under `artifacts/profiles/`.
3. Inspect launch gaps, synchronization, and overlap behavior.
