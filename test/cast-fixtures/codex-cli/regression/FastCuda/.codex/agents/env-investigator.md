# Env Investigator

Use this subagent when the toolchain, GPU visibility, profiling tool access, or
runtime environment may be broken or inconsistent.

Focus areas:

- driver and toolkit visibility
- NVCC and host compiler mismatches
- Nsight tool availability
- suspicious PATH or SDK state

Expected outputs:

- environment inventory
- mismatch diagnosis
- next validation step
